Origins

The Wi-Fi router was developed in 1997 by an Australian team consisting of:

  • John O’Sullivan
  • Terence Percival
  • Graham Daniels
  • Diethelm Ostry
  • John Deane

Alternative Names

Another name for a wireless router is the Wi-Fi router.

Functionality

A Wi-Fi router can be used to:

  • Access the internet
  • Connect to a private computer network

Wireless Bands

Some dual-band wireless routers operate on two frequency bands simultaneously:

  • 2.4 GHz band – up to 300 Mbit/s
  • 5 GHz band – up to 450 Mbit/s

Operating System

The most common operating system on such embedded devices is Linux.

SoftAP

A computer with the proper software can also run a SoftAP, which is essentially just a HotSpot.

Key Milestones

  • 1997: First Wi-Fi standard (802.11b) introduced.
  • 1999: Wi-Fi branding, as well as the formation of the Wi-Fi Alliance.
  • 2003: Wi-Fi 5 (802.11a/g) brings faster speeds.
  • 2009: Wi-Fi 4 (802.11n) introduces MIMO for better performance.
  • 2013: Wi-Fi 5 (802.11ac) offers up to 3.5 Gbps speeds.
  • 2018: Wi-Fi 6 (802.11ax) improves speed and efficiency.
  • 2023: Wi-Fi 7 promises even faster speeds, approaching 46 Gbps.

Network Management and Security

Handles traffic control, security, and settings.

Router Firmware

  • The software that controls router functions.
  • Examples: OpenWRT, DD-WRT, Tomato, or manufacturer firmware.

Firewall and Security

  • Built-in firewall to block malicious traffic.
  • Features like SPI (Stateful Packet Inspection) and DoS (Denial of Service) protection.

DHCP (Dynamic Host Configuration Protocol)

>
  • Assigns IP addresses to connected devices automatically.

VPN Support

>
  • Some routers support VPNs for encrypted browsing.

    • A VPN (Virtual Private Network) encrypts your internet traffic.

    • VPNS can help protect your security by replacing your IP address to a different part of the world
    • VPNS can also help bypass restrictions

How to Prevent WIFI Router Hacks & Secure your network

Change defualt login credentials

Changing your passwords to be unique and long can help prevent attacks on your router because hackers will be less prone to guessing your password :

  • Most routers use default usernames/passwords (e.g., admin/admin).
  • Change the admin password to a strong, unique one
  • Use long passwords with letters, numbers, and symbols

Use Strong WiFi Password & Encryption

Use WPA3 encryption (or WPA2 if WPA3 isn’t available) WPA3 is a security protocol used to protect wireless networks. It encrypts data transmitted over WiFi, preventing unauthorized access.

Enable Firewall & Security Features

Most routers have a built-in firewall. A firewall is a security system that can help monitors and controls network traffic based on predefined rules. It acts as a barrier between your private network and external threats, preventing unauthorized access and cyberattacks.
