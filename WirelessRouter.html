Wi-Fi Router Information

Origins

The Wi-Fi router was developed in 1997 by an Australian team consisting of:

  • John O’Sullivan
  • Terence Percival
  • Graham Daniels
  • Diethelm Ostry
  • John Deane

Alternative Name

Another name for a wireless router is the Wi-Fi router.

Functionality

A Wi-Fi router can be used to:

  • Access the internet
  • Connect to a private computer network

Wireless Bands

Some dual-band wireless routers operate on two frequency bands simultaneously:

  • 2.4 GHz band – up to 300 Mbit/s
  • 5 GHz band – up to 450 Mbit/s

Operating System

The most common operating system on such embedded devices is Linux.

SoftAP

A computer with the proper software can also run a SoftAP, which is essentially just a HotSpot.
